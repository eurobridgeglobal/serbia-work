# Работа в Сербии

Одностраничный статический сайт-воронка для сбора заявок от кандидатов, которые рассматривают работу в Сербии с официальным оформлением.

## Как открыть локально

Откройте файл `index.html` в браузере. Сайт не требует backend, сборки или установки зависимостей.

## Как разместить на GitHub Pages

1. Создайте репозиторий на GitHub.
2. Загрузите файлы `index.html`, `style.css`, `script.js`, `google-apps-script.js` и `README.md`.
3. В настройках репозитория откройте раздел `Pages`.
4. Выберите публикацию из ветки `main` и папки `/root`.
5. Сохраните настройки и дождитесь ссылки на сайт.

## Где менять тексты

Основные тексты сайта находятся в `index.html`. Можно менять заголовки, описания блоков, этапы процесса, список документов и текст формы.

## Где менять дизайн

Цвета, отступы, карточки, адаптивность и внешний вид формы находятся в `style.css`.

## Как подключить отправку в Google Таблицу

Логика формы находится в `script.js`. Замените значение `GOOGLE_SCRIPT_URL` на URL опубликованного Google Apps Script Web App.

Код для Google Apps Script находится в `google-apps-script.js`. Создайте Google Таблицу, откройте Apps Script, вставьте этот код и опубликуйте проект как веб-приложение с доступом для всех, у кого есть ссылка. Полученный URL вставьте в `script.js`.
